CSS Specificity

What is CSS Specificity

คือค่าระดับความสำคัญ โดย CSS จะเลือกสิ่งที่มีระดับความสำคัญมากกว่ามาแสดงก่อน โดยกรณีที่มีระดับความสำคัญเท่ากันก็จะเลือกอันที่อยู่ล่างกว่า

ความแตกต่างระหว่าง Specificity กับ Cascade

คือ Specificity จะเลือกคำสั่งที่มีค่าระดับความสำคัญมากกว่าส่วนแบบ Cascade จะเลือกคำสั่งที่อยู่ล่างกว่า

วิธีคำนวนหา CSS Specificity

วิธีคำนวน

ถ้าใส่ Style ให้ +1000

ถ้าใส่ Id ให้ +100

ถ้าใส่ Class ให้ +10

ถ้าใส่ Element ให้ +1

Example

ตัวอย่างการคำนวน

.box a {} คือเป็น Class .box และ มี Element a ด้วยทำให้มีค่าเท่ากับ 10+1 = 11

.page h1 #title {} คือเป็น Class .page, Element h1 และ Style#title ทำให้มีค่าเท่ากับ 10+1+100 = 111

div a {} คือเป็น Element div และ a ทำให้มีค่าเท่ากับ 1+1 = 2

div h1 a span {} คือเป็น Element div , h1 , a และ span ทำให้มีค่าเท่ากับ 1+1+1+1 = 4
