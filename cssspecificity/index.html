CSS Specificity

What is CSS Specificity

คือค่าระดับความสำคัญ โดย CSS จะเลือกสิ่งที่มีระดับความสำคัญมากกว่ามาแสดง โดยกรณีที่มีระดับความสำคัญเท่ากันก็จะเลือกอันที่อยู่ล่างกว่า

วิธีคำนวนหา CSS Specificity

วิธีคำนวน

ใส่ Style ให้ +1000

ใส่ Id ให้ +100

ใส่ Class ให้ +10

ใส่ Element ให้ +1

Example

ตัวอย่างการคำนวน

.box a {} คือ 10+1 = 11

.page h1 #title {} คือ 10+1+100 = 111

div a {} คือ 1+1 = 2

div h1 a span {} คือ 1+1+1+1 = 4
